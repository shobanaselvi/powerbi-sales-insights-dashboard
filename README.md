# 🛒 Blinkit Sales Analysis — Power BI Dashboard

## 🔍 Overview
An interactive Power BI dashboard analyzing Blinkit's grocery
sales data. Built to practice real-world business intelligence
and data storytelling using a popular e-commerce dataset.

## 🛠️ Tools & Skills Used
- **Tool:** Microsoft Power BI
- **Skills:** Data Modeling, DAX, KPI Tracking,
             Interactive Visuals, Business Insights

## 📌 Key Highlights
- Product-wise and category-wise sales breakdown
- Delivery performance and order trend analysis
- Revenue and profit KPI tracking
- Dynamic slicers for city/region level insights

## 🎯 Purpose
To demonstrate the ability to analyze real-world e-commerce
data and present actionable business insights using Power BI.
